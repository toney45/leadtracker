# 📌 Lead Tracker

A simple and lightweight Lead Tracker web application built with HTML, CSS, and JavaScript. Save important website links, manage your leads, and access them anytime using your browser's Local Storage.

## 🌐 Live Demo

https://leadtrackerss.netlify.app

---

## ✨ Features

- ➕ Save website URLs manually
- 📑 Save the current browser tab (Chrome Extension version)
- 💾 Data stored using Local Storage
- 🗑️ Delete all saved leads
- ⚡ Fast and lightweight
- 📱 Responsive design

---

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/toney45/leadtracker.git
```

### Open the project

Simply open `index.html` in your browser.

No installation required.

---

## 📖 How It Works

1. Enter a website URL.
2. Click **Save Input**.
3. The lead is stored in your browser using Local Storage.
4. Click a saved lead to visit it.
5. Double-click **Delete All** to clear every saved lead.
